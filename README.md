# EMACS-configs

## Summary

This is a copy of my EMACS configs.

## Motivation

It has two purposes:

* to facilitate the reinstallation of my system;

* to allow others to copy and understand and have a nice config for their EMACS.

## Install

The *home* directory contents should be copied to your local home directory */~* (if it is on an Unix like system).

## Details

The *.emacs* file has detailed comments of what each entry does and why some were disabled.

The *.gitignore* was designed to avoid adding:

* personal files;

    * history of commands on EMACS internal shell;

    * URLs used in EMACS;

    * Desktop with files that were open on the last session;

    * Filesets;

    * backup files;

    * autosave files;

    * session files;

    * etc;

* files used by EMACS:

    * lock files;

    * compiled files;

    * package manager files.

So do not wait for any personal information here.
